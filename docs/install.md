# Installation (macOS Excel)

1. Download the latest `ExcelMatlabLink.xlam` from GitHub Releases.
2. Open Excel.
3. Go to `Tools` -> `Add-ins`.
4. Click `Browse...` and select `ExcelMatlabLink.xlam`.
5. Enable macros if prompted.
6. A new ribbon tab should appear.

If the ribbon does not appear, see `docs/troubleshooting.md`.
If clipboard access is blocked, see `docs/troubleshooting.md#clipboard-access`.
